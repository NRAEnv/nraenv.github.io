Q*cert

This site hosts the (prettified) source code for the Q*cert certified query compiler. Q*cert is built and verified using the Coq proof assistant. A working compiler can be obtained by 'extracting' OCaml code from the source in Coq. The source documentation (generated using coqdoc) can be found here. Note that this is a large project, and not all of it is described in our paper submission.
